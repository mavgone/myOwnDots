# My Dotfiles

Personal Arch Linux + Hyprland setup.

## Components

* Hyprland
* Quickshell
* Kitty
* Neovim (LazyVim)
* Matugen
* Cava
* EasyEffects
* Spicetify
* SwayOSD
* GTK3 / GTK4
* Qt5ct / Qt6ct

## Notes

This setup is based on imperative-dots by ilyamiro and heavily modified.

Hyprland configuration is generated from:

* `hypr/settings.json`
* `hypr/templates/*`

Do not edit generated files directly:

* `hypr/config/keybindings.conf`
* `hypr/config/settings.conf`
* `hypr/config/monitors.conf`
* `hypr/config/env.conf`

Instead edit:

* `hypr/settings.json`
* files in `hypr/templates`
